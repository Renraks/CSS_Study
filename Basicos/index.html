Estilização

O CSS é usado basicamente para deixar o site mais bonito, enquanto o HTML trabalha como a base de como o site é separado o CSS cuidara de como o site aparenta para o usuario, embelezando-o

Por exemplo, usando o elemento style do CSS é possivel alterar como um certo paragráfo ou headline aparenta como a seguir

Este parágrafo está estilizado

Porém no caso anterior foi usado CSS dentro do próprio HTML, mas há como separa-lo e referencia-lo posteriormente para se ter uma maior clareza do código quanto mais ele crescer

A estilização tem 3 regras principais: seletor, propriedade e valor

  1. Seletor: Define qual elemento HTML será alterado pelo código em CSS
  2. Propriedade: Define oque será alterado
  3. Valor: Define para oque será alterado

Por exemplo se você utilizar com seletor o "p" todos os parágrafos serão alterados se você puxar o código CSS por um link

Outro exemplo é caso exista um ID, se um ID for utilizado como seletor, todos os atributos com aquele ID serão alterados independete da posição no documento

Este parágrafo possui o id "testeEstilo" usado no código para estiliza-lo

Os seletores de classe funcionam de uma maneira similar, a sua maior diferença dos ID's é que os ID's só podem ser aplicados uma vez por página, enquanto as classes podem ser usadas quantas vezes necessarias na página

Neste, e no próximo parágrafo, foi aplicado a classe "estileira" e seu comando do CSS

O seletor descendente é usado para aplicar o estilo dentro de um elemento e não há limite de quão profundo nele você pode ir, e usando este seletor, tudo que estiver fora da divisão selecionada, mesmo que tenha a mesma classe não usara o estilo

Este parágrafo está "profundo" na divisão

Este parágrafo está "profundo", mas fora da divisão

Em CSS para fazer comentários dentro do código utilizace os caracteres /* para abrir o comentario, e */ para fechá-lo

A aparencia final da página web é um resultado de diferentes regras de estilização, as 3 principais fontes da informação do estilo da página são:

  • A Stylesheet: O código feito pelo autor do site.
  • O browser: As definições de estilo padrões do browser usado.
  • O usuario: Estilos especificos usados pelo usuario.

Herança

A herança funciona de forma bem simples, tudo que estiver no meio de uma tag, é considerado filho de tal, dessa forma tudo dentro da tag /body/ é considerado filho desta tag, então se um código em CSS faz referência a uma tag pai, todas as tags filho herdaram o estilo da tag pai, a menos que sejam definidos novas cores para as tags filhos especificas depois.
