Decoração de Texto

A decoração de texto serve pra...................... decorar o texto :^O

Os valores que podem ser usados na decoração de texto são:

  • none: não muda nada do texto normal (aí é foda né kkkkkk)
  • inherit: herda as propriedades de seu elemento parente
  • overline: cria uma linha horizontal por cima do texto
  • underline: cria uma linha horizontal por baixo do texto
  • line-through: cria uma linha horizontal através do texto

Espaçamento de Letras

Os espaçamento de letras define qual vai ser a distancia, normalmente em pixels, de uma letra a a outra dentro de um parágrafo. Os valores a serem definidos podem ser normal, lenght e inherit que fazem, respectivamente, nada, decide uma distancia em px, pt, cm, mm, etc. e herda as configurações de seu elemento pai

Espaço em Branco

O espaçamento em branco mexe nos espaços que se da entre as palavras escritas em parágrafos no código, ele vai interagir com esass coisas, tem os valores normal, inherit e nowrap, o valor nowrap serve pra (porra nenhuma kkkkkkk) ignorar espaços em branco no texto assim como "enteres" dados no elemento "p" até que se encontre um "br/"

Outros valores são:

  • pre: o texto vai ser lido como foi escrito no código
  • pre-line: o texto vai ser lido ignorando espaços extras escritos no código, lendo as quebras de linha
  • pre-wrap: o texto vai ser lido tendo em vista quebras de linha e oque mais for necessario

Os textos a seguir vão estar todos escritos da mesma forma no código, mas terão códigos CSS diferentes
