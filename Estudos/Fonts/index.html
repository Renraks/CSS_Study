Fonts

Propriedade Familiar

As fontes possuem familias de proprieda que especificam a fonte a um elemento, e existem dois tipos de familia:

  • Familia Fonte: Especifica a familia da fonte (tipo Times New Roman ou Arial)
  • Familias Genéricas: Especifica familias de fontes visualmente similares (Como Serif ou Monospace)

Aqui um exemplo de algumas fontes:

Fonte Exemplo
serif Este texto está em serif
sansserif Este texto está em sansserif
monospace Este texto está em monospace
cursive Este texto está em cursive
fantasy Este texto está em fantasy

A propriedade de fonte precisa ter varios nomes de fonte como um tipo de sistema de segurança, para caso o computador do usuario não possua a fonte solicitada no código CSS, ele tentara usar o seguinte

Fontes cujos nomes possuem mais de uma palavra precisam ser descritos com " entre o nome


Propriedade de Tamanho

O tamanho das fontes pode ser alterado de varias formas, e uma dessas formas é utlizando palavras chave em seu código css como xx-small, small, medium, large, larger, etc.

Propriedade de Peso

A propriedade de peso controla o quão fino, ou grosso é o texto, os valores que podem ser usados são: normal, bold, bolder e lighter
