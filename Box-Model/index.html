O modelo de caixa em CSS


Entendendo oque é o modelo de caixa

Todos os elementos HTML estão dentro de caixas, e o modelo de caixa CSS representa o design e layout do site

Ele consiste de margens, bordas, preenchimento e o conteúdo.

As propriedades dessa caixa funcionam na seguinte maneira: topo, direita, fundo, e esquerda


Entendendo como funciona o modelo de caixa

Todos os elementos em uma página web são caixas

O CSS determina quão grande essas caixas são e onde elas ficam

A altura total da caixa é calculada da mesma forma que a largura.

Onde Altura= altura + preenchimento do topo + preenchimento do fundo + borda to topo + borda do fundo + margem do topo + margem do fundo


A propriedade de borda

A propriedade de borda do CSS permite a você editar as bordas de um documento HTML

As propriedades de borda no CSS representam, respectivamente:

  • Tamanho: define o tamanho da borda
  • Cor: define a cor da borda
  • Estilo: define o estilo da borda

As propriedades da borda podem ser alocadas separadamente, o comando border-width define o tamanho da borda separadamente.
